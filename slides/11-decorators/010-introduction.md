## Decorators

Decorators need to be enabled as a compiler option.

```json
{
    "compilerOptions": {
        "experimentalDecorators": true
    }
}
```

Special kind of declaration that can be attached to a `class`, `method`, `accessor`, `property` or `parameter` declaration.
