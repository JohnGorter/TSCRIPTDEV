ES2015 features

Default arguments

You can now use default parameters using =


// TypeScript

function hello(message = 'world') {
    console.log('Hello ' + message);
}


// Javascript

function hello(message) {
    if (message === void 0) {
        message = 'world';
    }
    console.log('Hello ' + message);
}
  • Default parameters should be placed after required parameters
