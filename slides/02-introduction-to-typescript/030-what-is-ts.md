## What is TypeScript

* *Syntactic sugar* for JavaScript
* Superset of EcmaScript vNext
* Types, classes, modules, decorators
* *Transpiles* to JavaScript
    * `ES3`
    * `ES5`
    * `ES2015`
    * `ES2016`
    * `ES2017`
    * `ESNext`
